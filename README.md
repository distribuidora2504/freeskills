## 🧩 Troubleshooting

Check these common issues if the software fails to start:

*   **Blocked File:** Windows might block the file because it comes from the internet. If this happens, right-click the setup file, select "Properties," and click "Unblock" near the bottom of the window before running it again.
*   **Missing Files:** If you see an error about missing files, reinstall the program. This often happens if the download stops early.
*   **Antivirus Software:** Some security programs might stop new software from loading. Temporarily pause your antivirus during installation if you have trouble.
